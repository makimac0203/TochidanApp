# TochidanApp
ダンスイベントのアーティストエントリー用のアプリケーションです。使用BGM、アーティスト写真、チーム構成などの主要な項目をアップロードして、管理者が容易に確認できるようにすることが目的です。


<br>
<br>
## 本番環境での使用技術
<br>
・AWS(ES2, RDS(PostgreSQL), S3)
・Nginx
・Ubuntu
・SendGrid
